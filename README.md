## Installation

CPaySDK is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'CPaySDK'
```
or

```ruby
pod 'CPaySDK', :subspecs => ['Core', 'Payment/CashApp',  'Payment/Klarna']
```
