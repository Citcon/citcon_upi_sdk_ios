# CPaySDK
> UPI mobile SDK for iOS.


## Installation

CPaySDK is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following line to your Podfile:


```ruby
pod 'CPaySDK'
```

or


```ruby
pod 'CPaySDK/Core'
pod 'CPaySDK/Ext/PPRiskMagnes'
pod 'CPaySDK/Ext/CardinalMobile'
```
